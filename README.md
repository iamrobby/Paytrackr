# PayTrackr

Invoice Aging Report + Automated Payment Reminders for Freelancers & Small Businesses.

Get paid faster with smart tracking and automatic reminders.

## Features

- Real-time Invoice Aging Report (Current, 1-30, 31-60, 61-90, 90+ days)
- Automated Email Reminders (7 days before & on due date)
- Add, Edit, Delete & Mark as Paid
- Client Email Support
- Beautiful Modern Dashboard
- Built with Next.js 15 + Supabase + Resend

## Tech Stack

- **Frontend**: Next.js 15 + TypeScript + Tailwind + shadcn/ui
- **Backend**: Supabase (PostgreSQL + Auth)
- **Emails**: Resend
- **Deployment**: Vercel
